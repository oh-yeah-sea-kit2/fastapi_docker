# fastapi_docker

## コマンド
- pipenv shell
- pipenv run python app.py
- pipenv install {library_name}

### pipenv環境でアプリ立ち上げ
- pipenv run uvicorn app:app --reload

### pytest実行
- pipenv run pytest
